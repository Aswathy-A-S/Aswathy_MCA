Notes on list in html

Ordered list

Ordered list are used to display a list of items in a specific order.The 'ol'tag is used to create an ordered list.Each item is enclosed in an 'li'tag.

  1. item 1
  2. item 2
  3. item 3

Unordered list

Unordered list are used to display a list of items in no specific order.The 'ul'tag is used to create an ordered list.Each item is enclosed in an 'li'tag.

  • item 1
  • item 2
  • item 3
